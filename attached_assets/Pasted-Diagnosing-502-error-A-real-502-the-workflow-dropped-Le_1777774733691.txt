What happened — three separate problems, three different causes:

1. First white screen (after I added dynamic import)
I wrote await import("./App") in main.tsx. That makes the entry point an async module. Vite's HMR doesn't handle async entry points correctly — it reloaded the module but the async boot never re-ran, so #root stayed empty. Fixed by reverting to a static import App from "./App" while keeping the try/catch around createRoot().render().

2. Second white screen (browser cache)
After fixing the code, your browser was still running the cached broken version from fix #1. The server was serving the correct file but the browser had the old one. Fixed by a hard refresh (Ctrl+Shift+R / Cmd+Shift+R).

3. 502 (Replit public proxy lost sync)
The 502 is not an app code issue. It happens when Replit's public domain proxy loses its connection to the workflow — usually after a workflow restart or when the proxy's routing table goes stale. The app was locally healthy the entire time (5000 → 200, 3001 → 200). Fixed by restarting the "Start application" workflow once, which forces Replit to re-establish the proxy connection to port 5000.
